# @vinciwu557/skill-demo

[![npm](https://img.shields.io/badge/npm-@vinciwu557/skill--demo-blue)](https://www.npmjs.com/package/@vinciwu557/skill-demo) [![License: MIT](https://img.shields.io/badge/license-MIT-green.svg)](./LICENSE)

示例 Pi 技能，展示 pi skill 的基本结构和使用方式。

## 功能

当用户提到"演示技能"或"demo skill"时使用此技能：

1. 先向用户打招呼
2. 解释这个技能的作用
3. 询问用户是否需要进一步帮助

## 安装

```bash
pi install npm:@vinciwu557/skill-demo
```

## 试用

```bash
pi -e npm:@vinciwu557/skill-demo
```
